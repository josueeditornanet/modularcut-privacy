Política de Privacidade

Modular Start  ·  br.com.modularstart.modularcut

Última atualização: maio de 2026

Resumo direto: o Modular Cut processa tudo localmente no seu dispositivo. O app funciona 100% offline. Não coletamos dados pessoais, não enviamos nada para servidores externos e não compartilhamos informações com terceiros.

1

Informações gerais

Esta Política de Privacidade descreve como o aplicativo Modular Cut, desenvolvido pela Modular Start, trata as informações dos usuários.

Ao utilizar o Modular Cut, você concorda com os termos descritos neste documento. Esta política aplica-se exclusivamente ao aplicativo Modular Cut para Android.

2

Dados acessados e como são usados

O Modular Cut foi projetado para funcionar com processamento 100% local e offline. Todos os dados permanecem no seu dispositivo:

DadoFinalidadeSai do dispositivo?
Áudio gravado pelo microfone Criar o roteiro de cenas (narração do próprio usuário) ❌ Nunca
Vídeos importados pelo usuário Corte e montagem local via FFmpeg ❌ Nunca
Vídeos cortados/montados Salvo no armazenamento interno do app ❌ Nunca
Dados do roteiro (marcações de tempo) Banco de dados SQLite local ❌ Nunca
3

Permissões do aplicativo

O Modular Cut solicita as seguintes permissões do Android, cada uma com finalidade específica e transparente:

PermissãoPor que é necessária
RECORD_AUDIO Gravar o roteiro de áudio pelo microfone do dispositivo (narração do usuário)
SYSTEM_ALERT_WINDOW Exibir o botão flutuante (overlay) sobre outros apps durante a gravação do vídeo
FOREGROUND_SERVICE Manter o serviço de overlay ativo enquanto o usuário grava em outro app
POST_NOTIFICATIONS Exibir notificação do serviço em foreground (exigido pelo Android 13+)
READ_MEDIA_VIDEO Importar os vídeos gravados da galeria (Android 13+)
READ/WRITE_EXTERNAL_STORAGE Compatibilidade com Android 10, 11 e 12
FOREGROUND_SERVICE_MEDIA_PROJECTION Permite ao usuário capturar conteúdo visual do dispositivo como material de referência para suas produções, quando optar por utilizar esse recurso
WAKE_LOCK Manter o áudio guia ativo durante gravação com tela desligada

Nenhuma permissão é usada para rastreamento, publicidade ou coleta de dados pessoais.

4

O que o aplicativo NÃO faz

  • Não envia áudios, vídeos ou quaisquer arquivos para servidores externos
  • Não se comunica com nenhum servidor — funciona 100% offline
  • Não coleta dados pessoais como nome, e-mail, localização ou contatos
  • Não possui sistema de cadastro ou login
  • Não acessa a galeria sem ação explícita do usuário (seleção manual de arquivo)
  • Não compartilha dados com terceiros de nenhuma natureza
  • Não exibe anúncios (sem AdMob, sem Meta Audience Network)
  • Não utiliza SDKs de analytics ou rastreamento (sem Firebase Analytics, sem Mixpanel)
  • Não monitora o comportamento do usuário dentro ou fora do app
5

Armazenamento e segurança dos dados

Todos os arquivos gerados pelo Modular Cut — áudios de roteiro, vídeos cortados, banco de dados de cenas — são salvos no diretório privado do aplicativo no dispositivo do usuário. Esses arquivos não são acessíveis por outros aplicativos e são removidos automaticamente quando o app é desinstalado.

A Modular Start não possui acesso remoto a nenhum dado do usuário. Não operamos servidores de sincronização, backup em nuvem ou qualquer infraestrutura que receba dados do dispositivo.

6

Compartilhamento com terceiros

O Modular Cut não compartilha, vende, licencia ou transfere dados do usuário para terceiros. Não há integrações com plataformas de publicidade, redes de análise ou qualquer serviço externo.

7

Crianças e menores de idade

O Modular Cut não é direcionado a crianças menores de 13 anos e não coleta intencionalmente informações de menores. Se você é responsável por uma criança e acredita que ela forneceu dados ao aplicativo, entre em contato conosco.

8

Alterações nesta política

A Modular Start pode atualizar esta Política de Privacidade periodicamente. Quando isso ocorrer, a data de "Última atualização" no topo da página será revisada. Recomendamos que os usuários consultem esta página ocasionalmente.
